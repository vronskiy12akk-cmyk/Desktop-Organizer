📂 Автоматическая сортировка файлов по типам (изображения, документы, архивы, музыка, видео, программы, прочее)

⚙️ Настраиваемые правила — вы сами решаете, какие расширения куда помещать

↩️ Отмена перемещений (Undo) — всегда можно вернуть файлы на место

🧾 Журнал действий для полной прозрачности

⏰ Автоматический режим — сортировка при запуске или по расписанию (где поддерживается)

🎯 Исключения — защита важных файлов и папок от перемещения

📊 Статистика — сколько и какие файлы были обработаны

🌍 Кроссплатформенность — работает на Windows, macOS, Linux

🎯 Что делает органайзер?
Просто укажите папку (по умолчанию — ваш рабочий стол), и программа:

Сканирует все файлы (кроме скрытых и исключённых)

Определяет категорию по расширению (или по содержимому для некоторых типов)

Создаёт в корне папки для каждой категории (если их нет)

Перемещает файлы в соответствующие папки

Ведёт журнал всех перемещений для возможности отмены

🧩 Настройка правил
Правила сортировки хранятся в файле config.json (или аналогичном) и выглядят так:
{
  "categories": {
    "Images": [".jpg", ".jpeg", ".png", ".gif", ".bmp", ".svg"],
    "Documents": [".pdf", ".doc", ".docx", ".txt", ".md", ".xls", ".xlsx"],
    "Archives": [".zip", ".rar", ".7z", ".tar", ".gz"],
    "Music": [".mp3", ".wav", ".flac", ".aac"],
    "Video": [".mp4", ".avi", ".mkv", ".mov", ".wmv"],
    "Programs": [".exe", ".msi", ".dmg", ".appimage", ".deb", ".rpm"],
    "Others": []
  },
  "excluded_folders": ["Desktop Organizer", "System"],
  "excluded_files": ["desktop.ini", ".DS_Store"],
  "auto_sort_on_start": true
}
